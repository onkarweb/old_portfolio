IamHungry

Project Description

This is a Java based food ordering app. Basically the user can order from three available items i.e. Salads, Sandwiches and Wraps. All the items are customizable and a reciept is printed at the checkout. This is a fun little app to show off with friends, well I am really proud that I made it :)

See the project repo here https://github.com/onkarweb/iamhungry

Project Details

  • Java Based
  • Prints receipts
  • Interactive UI
  • Uses file read/write logics
