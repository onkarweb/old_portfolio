The Unite

Project Description

This is a customized social media site for Sheridan College. It uses JavaScript library React.js for it's frontend and Node.js for the backend. Some of the features this project offers are: signup, signin, real time chat, friend requests, notifications, creating posts with pictures, uploading profile picture and liking the posts. This project serves the purpose to bring the student community at Sheridan much more closer.

See the live project here
(You might have to wait for 15s for the project to wake up. Since this is a prototype, you might have to refresh the page if you see any application error) https://theunite.herokuapp.com/login

Project Details

  • FrontEnd: React.js
  • BackEnd: Node.js
  • Database: PostgreSQL
  • Real time chat
  • Posting pictures
  • Send friend requests
  • Get notified for any updates with inbuilt notifications
