GameOn

Project Description

This is a Java Project made using Java fx library. It is a game project containing the popular games Sudoku and TicTacToe. It was my final project for my Object Oriented Programming II in second semester. You can have fun of login, registration, hall of fame, game saving options and many more in this project.

See the project repo here https://github.com/onkarweb/gameon

Project Details

  • Java Based Game
  • TicTacToe and Sudoku
  • Interactive UI
  • Uses Java FX
  • Uses file read/write for storage
