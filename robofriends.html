RoboFriends

Project Description

RoboFriends is a small fun web app made using React.js. It makes AJAX API calls to robohash.org and uses it to display dynamically created Robot images. It has a nice search box which filters out robots based on the search keyword. This app shows the power of React on a smaller scales

See the live project here https://onkarweb.github.io/robofriends/

Project Details

  • React.js based
  • Really Quick
  • AJAX API Calls
  • Autogenerated robo images
